## 2. Color System

### Philosophy
**Morandi monochromatic slate-blue palette** — 所有颜色来自同一色相家族 (240-260°), 通过明度和饱和度变体创建层次。没有任何纯黑或纯白，所有中性色都带有品牌色的微暖底调。

### Palette Tokens

```
Base Hue: 245-260° (slate-indigo)
```

## 3. Typography

### Font Stack
| Role | Font | Weight |
|------|------|--------|
| **Display** | `Outfit` | 600–700 |

| Token | Size | Line | Weight | Usage |
|-------|------|------|--------|-------|
| `text-xs` | 11px | 1.5 | 400 | Labels, captions |
| `text-sm` | 13px | 1.5 | 400 | Secondary text |

| `text-lg` | 16px | 1.5 | 600 | Section headings |
| `text-xl` | 18px | 1.4 | 600 | Page titles |
| `text-2xl` | 22px | 1.3 | 600 | Hero |

### Typography Rules
- Letter-spacing: `-0.01em` on body for crisp rendering
- Monospace: 12.5px for code blocks
- All headings: `font-display` (Outfit) for distinctive character
- No generic system fonts — Outfit + Inter combination creates identity

## 4. Spacing System

**4pt base grid** — all spacing derives from multiples of 4px.

| Token | Value | Usage |
|-------|-------|-------|
| `space-xs` | 4px | Icon gap, inline |
| `space-sm` | 8px | Tight siblings |
| `space-md` | 12px | Component padding |
| `space-lg` | 16px | Card padding |
| `space-xl` | 24px | Section gap |
| `space-2xl` | 32px | Page padding |
| `space-3xl` | 48px | Major section |

### Spacing Rules
- Use `gap` for sibling spacing (no margin hacks)
- Vertical rhythm: 24px base line
- Generous whitespace around CTAs (32–48px)
- Tight grouping for related elements (8–12px)

## 5. Effects & Materials

### Glassmorphism System

| Material | CSS | Usage |
|----------|-----|-------|
| **Standard Glass** | `backdrop-filter: blur(20px) saturate(120%)` <br/> `bg: oklch(18% 0.018 248 / 0.55)` | Sidebar, Cards |
| **Elevated Glass** | `backdrop-filter: blur(28px) saturate(130%)` <br/> `bg: oklch(22% 0.020 248 / 0.65)` | Dialogs, Modals |
| **Glass Border** | `oklch(36% 0.020 250 / 0.20)` | All glass surfaces |
| **Glass Highlight** | `oklch(100% 0 0 / 0.04)` | Top-edge light reflection |

### Watercolor Wash Background

Layered radial gradients create a subtle, drifting watercolor effect:

```
5 overlapping ellipses at different positions
Each: oklch tone with 7-12% opacity
Animation: 20s drift cycle (translate + scale)
```

Plus a **noise texture overlay** (SVG feTurbulence) at 2.5% opacity for paper-like tactility.

### Shadows

| Level | Value | Usage |
|-------|-------|-------|
| `sm` | `0 1px 3px oklch(0/0/0.12)` | Subtle depth |
| `md` | `0 4px 12px oklch(0/0/0.14)` | Cards |
| `lg` | `0 8px 24px oklch(0/0/0.16)` | Dropdowns |
| `xl` | `0 16px 48px oklch(0/0/0.20)` | Modals |
| `glow` | `0 0 40px oklch(62% 0.095 260 / 0.18)` | Accent elements |

### Easing Functions

| Name | Curve | Usage |
|------|-------|-------|
| `ease-out-expo` | `cubic-bezier(0.16, 1, 0.3, 1)` | Enter animations, hover lift |
| `ease-in-out-quart` | `cubic-bezier(0.76, 0, 0.24, 1)` | Page transitions |
| Duration fast | 150ms | Hover, focus |
| Duration base | 250ms | Toggle, expand |
| Duration slow | 400ms | Page entrance, dialogs |

## 7. Page Designs

### Chat Page (`currentPage === 'chat'`)
- Full chat interface with Header, Messages, Input
- Session context passes through all sub-components
- Permission dialogs triggered by tool calls

### Skills Page (SkillsPage.tsx)
- **Header**: Title + count subtitle + "创建技能" CTA
- **Search bar**: Full-width glass input with SearchIcon prefix
- **List**: Alternating-row cards with colored left border (success rate indicator)
- **Row content**: Name + version/type tags + description + success bar + delete button
- **Create Dialog**: Form with name, description, version, author, source
- **Detail Dialog**: Read-only skill metadata

### MCP Page (McpPage.tsx)
- **Header**: Title + description + "全部启动/停止" buttons
- **Grid**: 2-column server cards
- **Card content**:
  - Status dot: green pulse (running) / gray (stopped) / red (error)
  - Server name + error indicator
  - Description line
  - Command+args code block (monospace, ellipsis)
  - PID display
  - Toggle button (start/stop)
- **Auto-refresh**: 15s polling interval

### SubAgents Page (SubAgentsPage.tsx)
- **Header**: Title + count subtitle
- **Grouped list**: 3 groups (analysis / fix / delivery)
- **Group header**: Colored dot + uppercase label + count
- **Agent row**: Name + original ID + description + status Tag + Switch
- **Active**: Full opacity, colored left border, green tag
- **Inactive**: 50% opacity, transparent border, gray tag

### Settings Page (SettingsPage.tsx)
- **Section 1**: Login Configuration
  - Status indicator (CheckCircle / Loading / CloseCircle)
  - Environment variable form (API Key, Auth Token, Internet Env, Base URL)
  - CLI login instructions
- **Section 2**: Agent Configuration
  - Quick templates (4 preset cards)
  - Custom create/edit form (name, icon, color, permission, system prompt)
  - My Agents list (edit/delete)

## 9. AI Slop Prevention

### What We Avoid
- ❌ Pure black `#000` or white `#fff` — all colors are tinted
- ❌ Cyan/purple neon glow on dark backgrounds
- ❌ Inter as the only font (paired with Outfit for distinction)
- ❌ Cards inside cards (flat alternating rows instead)
- ❌ CSS `box-shadow` as the only depth cue (glass + elevation + border)
- ❌ `ease-out` / `ease-in-out` defaults (custom cubic-bezier curves)
- ❌ Rounded rectangles with colored border-left as the only visual trick
- ❌ Generic AI chat UI (centered input, minimal side panel)

### What Makes It Distinctive
- ✅ Morandi muted palette — no neon, no pure saturation
- ✅ Watercolor wash animated background — unique, not generic
- ✅ Glassmorphism with warm undertones, not cold tech cyan
- ✅ Outfit display font — distinctive, not Inter-everywhere
- ✅ Tinted shadows — warm cast, not pure black
- ✅ Monochromatic depth — hierarchy through lightness, not hue variety
- ✅ Tactile noise texture — subtle paper-like quality
- ✅ Semantic color accents in dusty tones (sage, amber, rose — not neon green/red/yellow)
